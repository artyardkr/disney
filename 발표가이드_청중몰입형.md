# 디즈니 발표 가이드 - 청중 몰입형 (일반 청중용)

## 발표 전략: "스토리텔링 × 데이터 × 재미"

### 핵심 원칙
1. **딱딱한 용어 → 쉬운 비유**
2. **숫자 나열 → 스토리로 포장**
3. **이론 설명 → 실제 경험 연결**
4. **일방향 → 쌍방향 (질문, 공감)**
5. **지루함 → 놀라움과 재미**
